Information/Help with AT-SCC

TO SEND: Under 'File' -> 'Select Mode' -> 'Send Mode'

Under 'Port Configurations', select desired settings from drop down menu(s)

Under 'Send/Receive Configurations', select desired type of data to send

Within the light blue panel, type in desired data to send

Select the log checkbox if you want to log the transfer

If you want to send and receive the data, configure the receive data type to match the send data type

Select the repeat checkbox if you want to send multiple times.

When ready, hit the 'START TRANSMISSION' button


TO RECEIVE: Under 'File' -> 'Select Mode' -> 'Receive Mode'

Under 'Port Configurations', select desired settings from drop down menu(s)

Under 'Send/Receive Configurations', select desired type of data to receive (ASCII/HEX will not work in this mode)

Select the log checkbox if you want to log the transfer

Select the repeat checkbox if you want to receive multiple times

When ready, hit the 'START TRANSMISSION' button


TO VIEW OR DELETE LOGS: Under 'File' -> 'Log File', select VIEW LOGS to open the log file, or select DELETE LOGS to delete the log file


TO RELOAD PORTS/PROGRAM: Under 'File' -> 'Reload'


OVERWRITING DATA: While repeating, please note that the max buffer for receiving is 100. If the buffer is exceeding, users have the option to either stop transmission automatically or overwrite the existing data in the output panel.

To change this, use the 'Overwrite Rx' toggle
