Information/Help with AT-SCC

TO SEND: Under 'File' -> 'Select Mode' -> 'Send Mode'

Under 'Port Configurations', select desired settings from drop down menu(s)

Under 'Send/Receive Configurations', select desired type of data to send

Within the light blue panel, type in desired data to send

Select the log checkbox if you want to log the transfer

If you want to send and receive the data, configure the receive data type to match the send data type

Select the repeat checkbox if you want to send multiple times.

When ready, hit the 'START TRANSMISSION' button
